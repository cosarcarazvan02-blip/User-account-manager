# User-account-manager
This is a simple C++ console application that allows users to input personal data and store it in a text file.
Features
Create a user account with:
Username
Personal ID
Number
Save user data to a file (Data.txt)
Read and display stored data from the file
How it works
The program asks for multiple users, collects their data, saves it to a file, and then reads and displays the saved information.
Technologies
C++
File handling (fstream)
Basic OOP (class, constructors, encapsulation)
